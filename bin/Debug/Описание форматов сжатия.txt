DEFAULT 
	- сжатие по умолчанию (как такового нет)
	Одноцветный тайл занимает: 
		3829 байт, PNG32 ~ 2.5 мс

FASTEST 
	- сжатие по умолчанию сравнивается со сжатым изображением
	с помощью FreeImage.DLL, сжатие PNG_Z_DEFAULT_COMPRESSION
	сохраняется файл с наименьшем размером
	Одноцветный тайл занимает: 
		2207 байт, PNG24 ~ 10 мс
	
FASTMAX
	- сжатие по умолчанию сравнивается со сжатым изображением
	с помощью FreeImage.DLL, сжатие PNG_Z_BEST_COMPRESSION
	сохраняется файл с наименьшем размером
	Одноцветный тайл занимает: 
		1798 байт, PNG24 ~ 34 мс
	
IMAGICK
	- сжатие по умолчанию сравнивается со сжатым изображением
	с помощью ImageMagic (Magick.NET-x86.dll), цветность PNG из соответствующего поля формы
	сохраняется файл с наименьшем размером
	(по умолчанию) показывает наиболее хорошие результаты сжатие/время
	Одноцветный тайл занимает: 
		1143 байт, PNG2 ~ 11 мс
	
OPTIMIZE
	- сжатие по умолчанию сравнивается со сжатым изображением
	с помощью Optimize.dll, параметры: -KeepInterlacing -KeepBackgroundColor -AvoidGreyWithSimpleTransparency
	сохраняется файл с наименьшем размером
	(максимальное сжатие) показывает хорошее сжатие, но работает очень долго
	Одноцветный тайл занимает: 
		922 байт, PNG2 ~ 390 мс
		
IMAGICK + FASTMAX
	- сжатие по умолчанию сравнивается со сжатым изображением
	с помощью ImageMagic (Magick.NET-x86.dll), цветность PNG из соответствующего поля формы и
	с помощью FreeImage.DLL, сжатие PNG_Z_BEST_COMPRESSION
	сохраняется файл с наименьшем размером
	Одноцветный тайл занимает: 
		1143 байт, PNG2 ~ 45 мс

IMAGICK + OPTIMIZE
	- сжатие по умолчанию сравнивается со сжатым изображением
	с помощью ImageMagic (Magick.NET-x86.dll), цветность PNG из соответствующего поля формы и
	с помощью Optimize.dll, параметры: -KeepInterlacing -KeepBackgroundColor -AvoidGreyWithSimpleTransparency
	сохраняется файл с наименьшем размером
	(максимальное сжатие) позволяет максимально сжать файл, по при этом работает очень долго
	Одноцветный тайл занимает: 
		922 байт, PNG2 ~ 405 мс
